6) Box & Violin (القيم الشاذة)

import seaborn as sns, numpy as np, pandas as pd
df = pd.DataFrame({"dept":np.repeat(["HR","IT","Sales"],100),

                                  np.random.normal(6,1.5,100)]})

7) Scatter (علاقة متغيرين) + Trend

plt.scatter(x,y); z = np.polyfit(x,y,1); p=np.poly1d(z); plt.plot(x,p(x))
plt.title("العلاقة والخط الاتجاهي"); plt.show()

18) رسوم متحركة بسيطة

import numpy as np, matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
x = np.linspace(0, 2*np.pi, 200)
fig, ax = plt.subplots(); ln, = ax.plot([], [])
def init(): ax.set_xlim(0, 2*np.pi); ax.set_ylim(-1,1); return ln,
def update(i): ln.set_data(x[:i], np.sin(x[:i])); return ln,
ani = FuncAnimation(fig, update, frames=len(x), init_func=init, blit=True)
plt.show()
📌 النتيجة: خط جيبي يتحرك تدريجيًا.
🎥 Animation

20) مشروع نهائي: Mini Dashboard للمبيعات

import pandas as pd, plotly.express as px
# بيانات مثال
df = pd.DataFrame({"date":pd.date_range("2025-01-01", periods=60, freq="D"),
                   "sales":(pd.Series(range(60)).rolling(7).mean().fillna(0)*3+50).round(1),
                   "region":["East","West","North","South"]*15})
# 1) خط زمني
fig1 = px.line(df, x="date", y="sales", title="اتجاه المبيعات")
# 2) أعمدة حسب المنطقة
fig2 = px.bar(df.groupby("region",as_index=False)["sales"].sum(), x="region", y="sales", title="المبيعات حسب المنطقة")
# 3) Scatter شهري (مثال)
df["month"]=df["date"].dt.to_period("M").astype(str)
fig3 = px.scatter(df, x="date", y="sales", color="region", title="تشتت المبيعات")
fig1.show(); fig2.show(); fig3.show()
📌 النتيجة: 3 رسوم تفاعلية تشكل لوحة مؤشرات مصغّرة.
🎥 Plotly Dashboard 🎥 نصائح التصميم
