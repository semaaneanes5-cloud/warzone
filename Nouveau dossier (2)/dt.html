2️⃣ Pandas – استيراد البيانات

import pandas as pd
df = pd.read_csv("data.csv")
print(df.head())
📌 النتيجة: عرض أول 5 صفوف من البيانات
🎥 فيديو Pandas

3️⃣ تنظيف البيانات

df = df.dropna()
df = df.drop_duplicates()
📌 النتيجة: بيانات نظيفة بدون فراغات أو تكرارات
🎥 فيديو التنظيف

4️⃣ استخدام Numpy

import numpy as np
arr = np.array([1,2,3,4,5])
print(arr.mean())
📌 النتيجة: 3.0 (المتوسط الحسابي)
🎥 فيديو Numpy

5️⃣ التعامل مع الملفات

# قراءة Excel
df = pd.read_excel("data.xlsx")

# قراءة JSON
df2 = pd.read_json("data.json")
📌 النتيجة: استيراد بيانات من مصادر مختلفة
🎥 فيديو قراءة البيانات

6️⃣ الإحصاء الوصفي

print(df.describe())
📌 النتيجة: المتوسط، الانحراف المعياري، القيم الصغرى والكبرى
🎥 فيديو الإحصاء

7️⃣ التصفية والفرز

print(df[df["age"] > 30])
print(df.sort_values("salary", ascending=False))
📌 النتيجة: بيانات الأشخاص فوق 30 سنة مرتبة حسب الراتب
🎥 فيديو التصفية

8️⃣ التعامل مع النصوص والتواريخ

df["date"] = pd.to_datetime(df["date"])
print(df["name"].str.upper())
📌 النتيجة: تحويل النصوص لأحرف كبيرة والتواريخ لصيغة موحدة
🎥 فيديو النصوص والتواريخ

9️⃣ GroupBy

print(df.groupby("department")["salary"].mean())
📌 النتيجة: متوسط الرواتب لكل قسم
🎥 فيديو GroupBy

🔟 Pivot Tables

pivot = pd.pivot_table(df, values="salary", index="department", columns="gender", aggfunc="mean")
print(pivot)
📌 النتيجة: جدول محوري يوضح الرواتب حسب القسم والجنس
🎥 فيديو Pivot

1️⃣1️⃣ Matplotlib

import matplotlib.pyplot as plt
df["salary"].hist()
plt.show()
📌 النتيجة: رسم هيستوغرام لرواتب الموظفين
🎥 فيديو Matplotlib

1️⃣2️⃣ Seaborn

import seaborn as sns
sns.boxplot(x="department", y="salary", data=df)
📌 النتيجة: رسم Boxplot للرواتب حسب الأقسام
🎥 فيديو Seaborn

1️⃣3️⃣ الارتباط Correlation

print(df.corr())
sns.heatmap(df.corr(), annot=True, cmap="coolwarm")
📌 النتيجة: مصفوفة ارتباط + Heatmap
🎥 فيديو Correlation

1️⃣4️⃣ Visualization متعدد الأبعاد

sns.pairplot(df, hue="department")
📌 النتيجة: رسم العلاقات بين جميع الأعمدة
🎥 فيديو Pairplot

1️⃣5️⃣ Dashboard مبسط

df["salary"].plot(kind="line")

plt.show()
📌 النتيجة: رسم خطي للرواتب عبر الزمن

1️⃣6️⃣ مشروع: تحليل مبيعات

📌 النتيجة: إجمالي المبيعات لكل منتج
🎥 فيديو المبيعات

1️⃣7️⃣ مشروع: تحليل نتائج الطلاب
