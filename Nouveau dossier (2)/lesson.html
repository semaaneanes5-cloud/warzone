HTML: ما هي ولماذا؟

HTML هي لغة ترميز تبني هيكل صفحة الويب. المتصفح يقرأ الوسوم tags ويحوّلها لعناصر مرئية.

هدف اليوم: فهم دور HTML ضمن الثلاثي (HTML هيكل — CSS تصميم — JS تفاعل).

تجهيز الأدوات

  • محرر: VS Code
  • متصفح حديث: Chrome / Edge / Firefox
  • ملف العمل: index.html
تلميح: افتح المجلد في VS Code واضغط Go Live إن كان لديك Live Server.
<!-- أول صفحة -->
<!DOCTYPE html>
<html lang="ar" dir="rtl">
  <head>
    <meta charset="utf-8">
    <title>مرحبًا HTML</title>
  </head>
  <body>
    ابدأ هنا 🚀
  </body>
</html>

هيكل الصفحة (Skeleton)

تعرف على الوسوم: <html>, <head>, <body>، وتعريف المستند <!DOCTYPE html>.

<!DOCTYPE html>
<html lang="ar">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>هيكل HTML</title>
  </head>
  <body>
    <header>ترويسة</header>
    <main>المحتوى</main>
    <footer>تذييل</footer>
  </body>
</html>

استخدم h1..h6 للعناوين، وp للفقرات، وstrong/em للتأكيد.

<h2>عنوان فرعي</h2>

مشروع 2: قائمة مهام (بدون باك‑إند)

<input id="todoIn"> <button id="add">أضف</button>
<ul id="list"></ul>
<script>
  const list = document.getElementById("list");
  document.getElementById("add").onclick = ()=>{
    const v = document.getElementById("todoIn").value.trim();
    if(!v) return;
    const li = document.createElement("li");
    li.textContent = v; list.appendChild(li);
    localStorage.setItem("todos", list.innerHTML);
  };
  window.onload = ()=>{ list.innerHTML = localStorage.getItem("todos")||"" };
</script>
