📱 تعلم Kotlin

كورس متكامل (20 درس) لتصبح مطور أندرويد محترف باستخدام Kotlin

1️⃣ مقدمة عن Kotlin

Kotlin لغة حديثة تدعمها Google لتطوير تطبيقات أندرويد.

fun main() { println("Hello Kotlin!") }

2️⃣ تثبيت Android Studio

تنزيل Android Studio وضبط Kotlin كمترجم افتراضي.

3️⃣ المتغيرات وأنواع البيانات

var و val للثوابت والمتغيرات.

val name="Anes"; var age=20

4️⃣ العمليات الحسابية والمنطقية

+ - * / % && || !

5️⃣ التحكم بالتدفق (if / when)

val grade=15; if(grade>=10) println("ناجح")

6️⃣ الحلقات (for / while)

for(i in 1..5){ println(i) }

7️⃣ الدوال (Functions)

fun sum(a:Int,b:Int)=a+b

8️⃣ Null Safety

كوتلن تمنع الأخطاء NullPointerException

var name:String?=null

9️⃣ المصفوفات والقوائم

val nums=listOf(1,2,3); println(nums[0])

🔟 المجموعات (Set / Map)

val map=mapOf("id" to 1,"name" to "Anes")

1️⃣2️⃣ الوراثة (Inheritance)

1️⃣3️⃣ الواجهات (Interfaces)

interface Clickable{ fun click() }

1️⃣4️⃣ الكائنات (Objects)

object DB{ val url="localhost" }

1️⃣5️⃣ البرمجة الكائنية المتقدمة

Encapsulation, Abstraction, Polymorphism

1️⃣6️⃣ التعامل مع الملفات

File("data.txt").writeText("Hello")

1️⃣7️⃣ التعامل مع JSON

استخدام مكتبة Gson أو kotlinx.serialization

1️⃣8️⃣ الكوروتينات (Coroutines)

للتعامل مع المهام غير المتزامنة.
